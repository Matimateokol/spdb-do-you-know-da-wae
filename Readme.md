
# How to run project for the first time?

DATABASE:
```cd database```
```./setup_db.sh```

FRONTEND:
```cd frontend-do-you-know-da-wae```
```yarn install```

```yarn run dev```

# How to run project each sequent time?

DATABASE:
```cd database```
```docker-compose start```

FRONTEND:
```cd frontend-do-you-know-da-wae```
```yarn run dev```
